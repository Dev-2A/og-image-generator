# 🖼️ OG Image Generator

프로젝트 이름, 설명, 기술 스택을 입력하면 SNS 공유용 OG(Open Graph) 카드 이미지를 자동 생성하는 도구입니다.

## ✨ 주요 기능

- 프로젝트 정보 입력 → OG 이미지 실시간 프리뷰
- 다양한 템플릿 (Minimal, Gradient, Card + Terminal Portfolio 테마 5종)
- 기술 스택 뱃지 자동 렌더링
- PNG 다운로드
- 브라우저만으로 동작 (백엔드 없음)

## 🛠️ 기술 스택

- React + Vite
- Satori (HTML → SVG)
- Tailwind CSS
- GitHub Pages
